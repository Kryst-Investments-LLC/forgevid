# SOC2 Controls

## Purpose
Document controls for security, availability, processing integrity, confidentiality, and privacy.

## Scope
All systems and processes impacting customer data.

## Key Controls
- Access control
- Change management
- Incident response
- Data encryption
- Monitoring and logging

## Review Instructions
- Review this policy quarterly and after any audit.
- Update controls and evidence as needed.
